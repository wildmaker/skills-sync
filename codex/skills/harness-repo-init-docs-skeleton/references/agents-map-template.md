# AGENTS.md Template (Map Style)

```markdown
# AGENTS.md

This file is a map, not an encyclopedia.

## Mission
- State the primary delivery objective.

## Navigation
- Architecture: `ARCHITECTURE.md`
- Design: `DESIGN.md`
- Plans: `PLANS.md`
- Reliability: `RELIABILITY.md`
- Security: `SECURITY.md`
- Design docs: `docs/design-docs/`
- Product specs: `docs/product-specs/`
- Execution plans: `docs/exec-plans/`

## Working model
1. Plan first.
2. Implement in small increments.
3. Validate with tests and runtime checks.
4. Update docs when behavior changes.

## Documentation policy
- Keep AGENTS concise.
- Store details under `docs/`.
```
