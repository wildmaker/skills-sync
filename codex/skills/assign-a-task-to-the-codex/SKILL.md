# assign-a-task-to-the-codex

## What this skill does
- 明确任务目标、范围、交付物与验收标准
- 在信息不足时先补齐需求再开工

## Constraints
- 不在任务未明确前开始执行
- 不猜测需求或验收标准

## Allowed commands
- None

## Steps
1. 复述任务目标与期望交付物。
2. 询问缺失信息（范围、期限、约束、验收标准）。
3. 获得确认后再开始执行任务。
